### Overview

**ABC Company** is a fictional business website I built to showcase modern web development practices.  
The project focuses on performance, responsiveness, and clean design.

### Tech Stack

- **Svelte & SvelteKit** — for fast, reactive frontend development
- **Tailwind CSS** — for elegant and consistent UI styling
- **SendGrid API** — for reliable contact form email handling
- **Sanity CMS** — for flexible, headless content management
- **Vercel** — for seamless deployment and hosting

### Features

- Fully responsive layout optimized for all devices
- Custom contact form with email delivery
- SEO-friendly structure and fast load times
- Modular content managed directly from Sanity CMS

### Outcome

This project strengthened my frontend and deployment workflow skills.  
I gained practical experience integrating APIs and CMS systems into a modern web stack.
