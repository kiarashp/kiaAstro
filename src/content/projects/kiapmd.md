### Overview

This **personal portfolio website (Astro)** was built to showcase my projects and skills in a clean, performant, and SEO-friendly way.  
Using Astro.js, I implemented a component-based architecture and Markdown content collections, leveraging **islands architecture** for fast interactivity without sacrificing performance.

### Tech Stack

- **Astro.js** — for static site generation and modern frontend architecture  
- **Tailwind CSS & DaisyUI** — for responsive and elegant UI design  
- **Markdown & Astro Content Collections** — for easy content management  
- **Islands Architecture** — for optimal performance and interactive components  

### Features

- Fully responsive layout optimized for all devices  
- SEO-friendly structure with fast page load times  
- Modular content management using Markdown collections  
- Interactive components using Astro islands for minimal JS  
- Elegant UI powered by DaisyUI and Tailwind CSS  

### Outcome

This project allowed me to **strengthen my skills in Astro.js, frontend architecture, and responsive design**.  
It also gave me hands-on experience managing content efficiently while building a fast, modern, and SEO-friendly personal website.
