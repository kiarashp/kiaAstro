### Overview

**Alvan Textile** is a modern industrial website built for a factory specializing in **fabric and garment coloring processes**.  
The primary objective was to create a **clear, fast, and trustworthy online presence** suitable for an industrial and manufacturing-focused audience.

The website is **fully static**, built with Astro.js, and deployed on **Vercel**, eliminating the need for traditional server management while ensuring excellent performance and reliability.

### Tech Stack

- **Astro.js** — static-first framework for high-performance websites  
- **TypeScript** — type-safe development and maintainable codebase  
- **Tailwind CSS** — utility-first styling for responsive and consistent design  
- **DaisyUI** — ready-to-use accessible components and theming  
- **Vercel** — serverless deployment and global edge delivery  

### Features

- Clean industrial UI tailored for manufacturing and factory presentation  
- Fully static architecture for maximum speed and stability  
- Responsive layout optimized for desktop, tablet, and mobile  
- SEO-friendly structure with semantic HTML  
- Optimized assets and images for fast loading  
- Simple deployment pipeline via Vercel without server configuration  
- Maintainable and scalable frontend architecture  

### Outcome

The project delivers:

- Excellent performance and fast page loads  
- Strong SEO foundations suitable for industrial discovery  
- A professional and credible digital presence for a textile factory  
- A maintainable, future-proof frontend built with modern tooling  

This project demonstrates practical expertise in **Astro.js**, **industrial website design**, and **performance-focused frontend development** using a modern, serverless deployment approach.
