### Overview

**Thermo Group** is a static industrial website designed to present heating equipment, ceramic materials, and thermal components in a professional and clear format.  
The goal was to build a fast, accessible, and SEO-friendly site using modern, maintainable tools and best practices.

### Tech Stack

- **Astro.js** — delivering a lightweight, high-performance static architecture  
- **Tailwind CSS** — for fast, consistent, and fully responsive styling  
- **DaisyUI** — providing ready UI themes with built-in dark/light mode  
- **Markdown + Astro Collections** — enabling structured, scalable content management  
- **Nginx + Cloudflare** — for optimized deployment, caching, and global performance  

### Features

- Fully static, high-performance Astro.js build  
- Light/Dark mode powered by DaisyUI  
- Product categories for multiple industrial sectors  
- Markdown-driven content for easy management  
- Clean typography and accessible UI  
- SEO-optimized structure and metadata  
- Responsive design across all devices  

### Outcome

This project strengthened my experience in **static site development**, **content architecture**, and **performance-driven frontend engineering**.  
By combining Astro.js, Tailwind, DaisyUI, and Markdown, I created a fast, maintainable platform ideal for technical product catalogs and industrial businesses.
