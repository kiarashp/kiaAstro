### Overview

This **personal portfolio website (WordPress & Elementor)** was built to showcase my projects and skills using a flexible and user-friendly CMS.  
By leveraging **Elementor** along with the **Hello Elementor theme**, I was able to create a fast, responsive, and visually appealing design without writing custom code.

### Tech Stack

- **WordPress** — for robust content management and easy updates  
- **Elementor Page Builder** — for drag-and-drop responsive layouts  
- **Hello Elementor Theme** — lightweight foundation for fast performance  
- **SEO Optimization** — structured content for better search engine visibility  

### Features

- Fully responsive design optimized for desktops, tablets, and mobile devices  
- Drag-and-drop visual editing for rapid content updates  
- SEO-friendly structure and fast-loading pages  
- Clear showcase of projects, skills, and portfolio items  
- Easy maintenance and content management without developer intervention  

### Outcome

This project strengthened my skills in **WordPress development, Elementor design, and SEO best practices**.  
It also provided hands-on experience creating a professional portfolio website that is easy to manage and visually engaging.
